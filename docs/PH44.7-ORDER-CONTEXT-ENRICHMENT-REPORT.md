## 3. Fichiers modifies

### `ai-assist-routes.ts`

**OrderItem** ‚Äî ajout `sku`:
```typescript
interface OrderItem {
  title: string;
  quantity: number;
  price?: number;
  asin?: string;
  sku?: string;    // PH44.7
}
```

**OrderContext** ‚Äî ajout `marketplaceId`, `marketplaceLabel`, `deliveryWindow`:
```typescript
interface OrderContext {
  // ... existants ...
  marketplaceId?: string;           // PH44.7
  marketplaceLabel?: string;        // PH44.7
  deliveryWindow?: {                // PH44.7
    earliest: string;
    latest: string;
  };
}
```

**SQL SELECT** ‚Äî ajout `raw_data`:
```sql
SELECT id, external_order_id, status, order_date, updated_at,
  fulfillment_channel, carrier, tracking_code, total_amount, currency,
  delivery_status, products, raw_data    -- PH44.7: raw_data added
FROM orders WHERE tenant_id = $1 AND external_order_id = $2
```

**loadConversationContext** ‚Äî extraction raw_data:
```typescript
const rawOrder = order.raw_data?.order;
const mpId = rawOrder?.MarketplaceId || undefined;
let deliveryWindow = undefined;
if (rawOrder?.EarliestDeliveryDate && rawOrder?.LatestDeliveryDate) {

    earliest: rawOrder.EarliestDeliveryDate,
    latest: rawOrder.LatestDeliveryDate,
  };
}
```
