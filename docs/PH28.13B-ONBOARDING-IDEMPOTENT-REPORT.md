# PH28.13B - Onboarding Idempotent Report

**Date:** 2026-02-02
**Status:** ✅ COMPLETED
**Environments:** DEV + PROD

---

## Objectif

Après un signup (OTP ou Google), si un tenant existe déjà pour l'utilisateur :
- `/onboarding` ne doit **pas** tenter de créer un second tenant
- Redirection automatique vers le dashboard
- Aucun écran d'erreur "création tenant échoue car existe"

---

## Modifications Effectuées

### 1. Backend - Endpoint Idempotent

**Fichier:** `keybuzz-api/src/modules/auth/tenant-context-routes.ts`

Le endpoint `POST /tenant-context/create-signup` a été rendu idempotent :

```typescript
// PH28.13B: Check if user already has a tenant as owner (idempotent check)
const existingTenantResult = await pool.query(
  `SELECT t.id, t.name, tm.is_trial, tm.trial_ends_at
   FROM tenants t
   JOIN user_tenants ut ON t.id = ut.tenant_id
   LEFT JOIN tenant_metadata tm ON t.id = tm.tenant_id
   WHERE ut.user_id = $1 AND ut.role = 'owner'
   ORDER BY t.created_at DESC
   LIMIT 1`,
  [userId]
);

if (existingTenantResult.rows.length > 0) {
  const existingTenant = existingTenantResult.rows[0];
  console.log(`[TenantContext] User ${email} already has tenant ${existingTenant.id}, returning existing (idempotent)`);
  
  // Set as current tenant
  await pool.query(
    `INSERT INTO user_preferences (user_id, current_tenant_id, updated_at)
     VALUES ($1, $2, NOW())
     ON CONFLICT (user_id) DO UPDATE SET current_tenant_id = $2, updated_at = NOW()`,
    [userId, existingTenant.id]
  );

  return reply.send({
    success: true,
    tenant: { ... },
    currentTenantId: existingTenant.id,
    alreadyExisted: true,  // Flag idempotent
  });
}
```

### 2. Frontend - Preflight Check

**Fichier:** `keybuzz-client/app/onboarding/page.tsx`

La page `/onboarding` effectue maintenant un preflight check au chargement :

```typescript
const checkTenant = async () => {
  try {
    const response = await fetch(API_URL + '/tenant-context/tenants', {
      credentials: 'include',
      headers: {
        'X-User-Email': session?.user?.email || '',
      },
    });
    if (response.ok) {
      const data = await response.json();
      if (data.tenants && data.tenants.length > 0) {
        const tenant = data.tenants[0];
        console.log('[Onboarding] Tenant already exists, redirecting:', tenant.name);
        
        // Sélectionner le tenant
        await fetch('/api/auth/select-tenant', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          credentials: 'include',
          body: JSON.stringify({ tenantId: tenant.id }),
        });
        
        // Rediriger vers dashboard
        router.replace('/');
        return;
      }
    }
  } catch (err) {
    console.error('Failed to fetch tenant:', err);
  } finally {
    setIsLoading(false);
  }
};
```
