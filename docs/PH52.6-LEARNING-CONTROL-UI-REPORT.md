## 1. Objectif

Creer une interface visuelle pour le controle de l'apprentissage IA (PH52.5).
Aucune modification de la logique backend (hormis la securite PROD du POST).

## 2. Interface Utilisateur

### Page Parametres > Onglet "Intelligence Artificielle"

La section **IA Conversationnelle** est ajoutee sous la section existante (mode IA, limites, kill switch).

#### 3 Modes

| Mode | Description | Backend |
|---|---|---|
| **Standard** | L'IA suit les regles SAV. Pas d'adaptation automatique. | collect=true, apply=false |
| **Adaptatif** | L'IA apprend des reponses humaines et adapte ses suggestions. | collect=true, apply=true |
| **Expert** | Mode adaptatif avec controles avances. | collect=true, apply=true |

#### Toggle Apprentissage

> Apprentissage automatique
> [ ] Autoriser l'apprentissage a partir des reponses humaines

Quand OFF : collect=false, apply=false — PH51 et PH52 sont completement desactives.

### Badge Mode IA

Dans le panneau **Aide IA** (`AISuggestionSlideOver`), un badge affiche le mode actif :

```
IA : Standard        (gris)
IA : Adaptative      (indigo)
```

Le badge apparait a cote du compteur KBActions dans le header du slide-over.

## 4. Securite

| Env | GET | POST |
|---|---|---|
| DEV | Tout membre du tenant | Tout membre du tenant |
| PROD | Tout membre du tenant | Owner ou Admin uniquement |

Le POST en PROD retourne 403 pour les agents (role `agent`).

## 5. Flux Utilisateur

```
1. User va dans /settings > onglet "Intelligence Artificielle"
2. Scrolle vers la section "IA Conversationnelle"
3. Selectionne un mode (Standard / Adaptatif / Expert)
   → POST /api/ai/learning-control → backend → DB + invalidate Redis
4. Active/desactive le toggle "Apprentissage automatique"
   → Meme flux
5. Dans l'Inbox, ouvre le panneau "Aide IA"
   → Badge "IA : Standard" ou "IA : Adaptative" visible dans le header
```
