## Modèle Octopia

**Mode : Délégation**

Le vendeur délègue son accès Cdiscount à KeyBuzz via le Seller Portal Octopia.

```
1. Vendeur → Seller Portal Octopia
2. Vendeur délègue accès à "KeyBuzz" (agrégateur)
3. KeyBuzz vérifie la délégation
4. Messages Cdiscount → KeyBuzz
```
