## Fichiers crees

### 1. `src/features/inbox/utils/messageClassifier.ts` (2463 bytes)

**Type export** : `InboundMessageType = 'CLIENT' | 'AMAZON_AUTO' | 'SYSTEM'`

**Logique de classification** :

- Score-based : combine signaux contenu + signaux sender
- Seuil : score >= 2 pour AMAZON_AUTO (conservatif, zero faux positif)
- 16 patterns de contenu Amazon (FR, EN, IT, ES)
- 4 patterns sender (no-reply, noreply, seller central, amazon)
- SYSTEM : detection via messageSource ou conversationType
- SUPPLIER : exclu de la classification (retourne CLIENT)

**Fonctions utilitaires** :

- `getMessageTypeLabel()` : retourne le label FR
- `getMessageTypeColor()` : retourne les classes Tailwind (bg, text, border)
