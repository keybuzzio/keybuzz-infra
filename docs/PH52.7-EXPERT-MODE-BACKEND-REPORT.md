## Objectif

Ajouter un 3e etat backend reel pour distinguer Standard, Adaptatif et Expert.

## Migration SQL

```sql
ALTER TABLE tenant_ai_learning_settings
ADD COLUMN IF NOT EXISTS learning_advanced_ui_enabled BOOLEAN NOT NULL DEFAULT false;
```

Idempotente, aucune suppression, aucun changement destructif.

## Mapping des 3 modes

| Mode | collect_enabled | apply_enabled | advanced_ui_enabled |
|------|----------------|---------------|---------------------|
| Standard | true | false | false |
| Adaptatif | true | true | false |
| Expert | true | true | true |
| Off (toggle) | false | false | false |

### Fonction `computeMode()`

```typescript
function computeMode(collect, apply, advancedUi) {
  if (!collect || !apply) return 'standard';
  if (advancedUi) return 'expert';
  return 'adaptive';
}
```
