## Etape 5 — Logique bouton pause

### Decision

Le bouton "Mettre KeyBuzz en pause" (kill_switch) a ete **supprime**.

### Justification

- `kill_switch` (ai_settings) = ancien systeme
- `is_enabled` (autopilot_settings) = nouveau systeme
- Avoir les deux = deux concepts concurrents pour la meme action

### Ce qui reste

- **`is_enabled`** dans AutopilotSection : toggle unique d'activation
- **`auto_disabled`** dans AISettingsSection : alerte systeme de securite (si l'IA se desactive automatiquement apres erreurs consecutives) avec bouton "Reactiver"
