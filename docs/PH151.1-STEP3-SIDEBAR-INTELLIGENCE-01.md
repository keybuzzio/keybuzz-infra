## Objectif

Ajouter de l'intelligence dans la sidebar droite (OrderSidePanel) :

1. **topSlot** : nouveau prop `React.ReactNode` affiché en haut du panneau, avant le contenu existant
2. **AICaseSummary dans la sidebar** : le résumé du dossier est injecté via topSlot
3. **Collapsible** : le panneau peut être réduit/étendu via un bouton

## Modifications détaillées

### OrderSidePanel.tsx

**Props ajoutés :**

- `topSlot?: React.ReactNode` — contenu injecté en haut du panneau
- `collapsible?: boolean` — active le mode réductible

**State ajouté :**

- `const [collapsed, setCollapsed] = useState(false)`

**Comportements :**


| État                       | Rendu                                                   |
| -------------------------- | ------------------------------------------------------- |
| `collapsed && collapsible` | Barre fine (w-10) avec bouton PanelLeftOpen pour expand |
| `isOpen && !collapsed`     | Panneau complet (w-80) avec topSlot + contenu existant  |
| `!isOpen`                  | `null` (inchangé)                                       |


**Zones touchées :**


| Zone                     | Modification                                          | Risque                      |
| ------------------------ | ----------------------------------------------------- | --------------------------- |
| Interface props          | +2 props optionnels                                   | Aucun (backward-compatible) |
| Fonction signature       | +2 params destructurés                                | Aucun                       |
| Lucide import            | +PanelRightOpen, PanelLeftOpen                        | Aucun                       |
| Early return (`!isOpen`) | +bloc collapsed                                       | Faible                      |
| Empty state              | +collapse button + topSlot                            | Faible                      |
| Main return header       | +collapse button dans div flex + topSlot après header | Faible                      |


**Aucune section existante supprimée ou déplacée.**

### InboxTripane.tsx

Ajout de 2 props sur l'appel `<OrderSidePanel>` :

```tsx
topSlot={selectedConversation ? <AICaseSummary conversation={selectedConversation} /> : undefined}
collapsible
```
