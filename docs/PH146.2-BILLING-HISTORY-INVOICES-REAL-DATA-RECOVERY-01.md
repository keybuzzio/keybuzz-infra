## 3. Cause racine

### /channels — quota faux

**Ligne 378** : `billingCompute.channelsIncluded` retourne uniquement les canaux inclus dans le plan (5), pas le total disponible (inclus + addons = 11).

### /billing/history — données factices

La page entière avait été créée comme maquette statique avec :

- `MOCK_INVOICES: any[] = []` (ligne 8)
- `UPCOMING_INVOICE = { date: '01 Feb 2026', description: 'Plan Pro - Mensuel', amount: '297,00 EUR' }` (lignes 10-14)
- Banner "Paiements bientôt disponibles" (lignes 70-86)
- `useCurrentPlan()` importé mais jamais utilisé

---

## 4. Source de vérité retenue

`**useCurrentPlan()` + `useChannelLimit()`** (hooks React alimentés par `/api/billing/current`)

Calcul du total canaux :

```
total disponible = channelsIncluded + channelsAddonQty
                 = 5 + 6 = 11
```

Calcul du total mensuel :

```
total mensuel = planInfo.price + (channelsAddonQty × capabilities.channelPrice)
              = 497 + (6 × 50) = 797 EUR HT/mois
```
