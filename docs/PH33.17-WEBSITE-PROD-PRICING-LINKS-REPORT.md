# PH33.17 — Pricing links → PROD (client.keybuzz.io)

## Objectif
Basculer tous les CTA de `keybuzz.pro/pricing` de `client-dev.keybuzz.io` vers `client.keybuzz.io` (PROD).

## Changements

### Fichier modifie
`keybuzz-website/src/app/pricing/page.tsx`

### Modifications
1. **URL fallback** : `client-dev.keybuzz.io` → `client.keybuzz.io`
2. **Parametre `cycle`** ajoute a tous les liens CTA : `&cycle=monthly` (defaut) / `&cycle=yearly` (toggle)
3. **Toggle mensuel/annuel** ajoute avant les cartes pricing
4. **Prix dynamique** : affichage du prix barre + prix remise (-20%) quand annuel selectionne
5. **Build arg** : `NEXT_PUBLIC_CLIENT_APP_URL=https://client.keybuzz.io` passe au build Docker

### Aucun hardcode
- L'URL vient de `process.env.NEXT_PUBLIC_CLIENT_APP_URL` avec fallback `client.keybuzz.io`
- Le toggle `isYearly` est un state React qui modifie dynamiquement le lien au render
