**Date:** 2026-02-03
**Status:** ✅ COMPLETED
**Environments:** DEV + PROD

## Objectif

Ajouter sur le flow OTP :
- Décompte 45s avant resend
- Lien "Renvoyer le code" actif après expiration
- Message "Vous n'avez rien reçu ? Vérifiez vos spams."
- Toast "Nouveau code envoyé." après resend
- Reset décompte après resend

---

## UX Implémentée

### Pendant le décompte (0-45s)
```
Renvoyer le code dans 43s
```
- Lien désactivé (grisé)

### Après expiration (0s)
```
[Renvoyer le code]
Vous n'avez rien reçu ? Vérifiez vos spams.
```
- Lien actif (cliquable)

### Pendant l'envoi
```
Envoi...
```
- Bouton désactivé avec spinner

### Après resend réussi
```
Nouveau code envoyé.
Renvoyer le code dans 45s
```
- Message de confirmation
- Décompte reset

### En cas de rate-limit (429)
```
Veuillez patienter avant de demander un nouveau code.
```

```typescript
// PH28.15: Resend countdown
const [resendCooldown, setResendCooldown] = useState(0);
const [resendLoading, setResendLoading] = useState(false);
const [resendMessage, setResendMessage] = useState<string | null>(null);
const cooldownRef = useRef<NodeJS.Timeout | null>(null);
const countdownStartedRef = useRef(false);

// Countdown timer effect
useEffect(() => {
  if (step === 'code' && !countdownStartedRef.current) {
    countdownStartedRef.current = true;
    setResendCooldown(45);
  }
  if (step !== 'code') {
    countdownStartedRef.current = false;
  }
  if (resendCooldown > 0) {
    cooldownRef.current = setTimeout(() => {
      setResendCooldown(prev => prev - 1);
    }, 1000);
  }
  return () => {
    if (cooldownRef.current) clearTimeout(cooldownRef.current);
  };
}, [step, resendCooldown]);

// Resend function
const handleResendCode = async () => {
  if (resendCooldown > 0 || resendLoading) return;
  // ... call same endpoint as initial send
  setResendMessage('Nouveau code envoyé.');
  setResendCooldown(45);
};
```
