## 3. Comment identifier la "suggestion IA" vs "reponse humaine"

### Suggestion IA
- Source : `ai_action_log` WHERE `action_type = 'AI_DECISION_TRACE'`
- Le `payload.savScenario` indique le type de probleme detecte
- Le `payload.nextBestStep` indique l'action recommandee
- Le `payload.allowedActions` liste les actions autorisees

### Reponse humaine finale
- Source : `messages` WHERE `direction = 'outbound'` et `conversation_id` match
- Le `body` contient le texte exact envoye par l'agent humain
- On prend le DERNIER message outbound apres le dernier AI trace

### Detection de "correction"
La comparaison se fait par detection de mots-cles d'action dans le texte humain :
- **Remboursement** : rembours*, refund*, credit*
- **Remplacement** : remplac*, envoy* nouveau*, send* new*
- **Garantie** : garantie, warranty, fournisseur, supplier, SAV
- **Investigation** : enquete, investigation, verif*, track*, suivi
- **Information** : information, confirm*, precis*, detail*

On compare ensuite le type d'action detecte dans la reponse humaine vs le `nextBestStep` de l'IA.
