**Contenu :**

```typescript
import { NextResponse } from 'next/server';

export const dynamic = 'force-dynamic';

const CHANNELS_REGISTRY = [
  { id: 'amazon', label: 'Amazon Seller Central', ... status: 'available' },
  { id: 'octopia', label: 'Octopia (Cdiscount)', ... status: 'available' },
  { id: 'fnac', label: 'Fnac Marketplace', ... status: 'coming_soon' },
  { id: 'email', label: 'Email', ... status: 'coming_soon' }
];

export async function GET() {
  return NextResponse.json(CHANNELS_REGISTRY);
}
```

### Frontend

| Fichier | Modification |
|---------|--------------|
| `app/channels/page.tsx` | Fetch registry au lieu de hardcode |
| `middleware.ts` | Ajout `/api/channels/registry` à PUBLIC_ROUTES |

**Changements channels/page.tsx :**

```typescript
// Avant (hardcodé)
const otherChannels = [
  { id: "fnac", name: "Fnac Marketplace", ... },
  { id: "email", name: "Email", ... },
];

// Après (dynamique via registry)
interface ChannelRegistry {
  id: string;
  label: string;
  description: string;
  logo: string;
  status: 'available' | 'coming_soon';
}
const [channelsRegistry, setChannelsRegistry] = useState<ChannelRegistry[]>([]);

// Fetch on load
const regRes = await fetch('/api/channels/registry');
if (regRes.ok) {
  const registry = await regRes.json();
  setChannelsRegistry(registry);
}

// Filter for coming_soon
const otherChannels = channelsRegistry.filter(ch => ch.status === 'coming_soon');
```

## Multi-tenant

La registry est **globale produit**, pas tenant-scoped :
- Pas de `tenantId` dans l'endpoint
- Données identiques pour tous les tenants
- Le statut connecté/non-connecté reste tenant-scoped (via `/api/octopia/status`, `/api/amazon/status`)
