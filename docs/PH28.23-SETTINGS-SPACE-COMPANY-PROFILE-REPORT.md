## Amélioration Requise

### Pour intégrer l'API Profile :

1. **Au chargement:**
```typescript
useEffect(() => {
  if (!tenantId) return;
  
  // Fetch from API
  fetch(`/api/tenant-context/profile/${tenantId}`)
    .then(res => res.json())
    .then(data => {
      if (data.profile) {
        setProfile({
          companyName: data.profile.companyName || defaultProfile.companyName,
          email: data.profile.supportEmail || defaultProfile.email,
          // ...
        });
      }
    });
}, [tenantId]);
```

2. **À la sauvegarde:**
```typescript
const saveSettings = useCallback(async () => {
  // Save to API
  await fetch(`/api/tenant-context/profile/${tenantId}`, {
    method: 'PUT',
    body: JSON.stringify({ companyName, supportEmail, country }),
  });
  
  // Also save to localStorage for offline/cache
  localStorage.setItem(STORAGE_KEY, JSON.stringify(data));
}, [...]);
```

## Conclusion

**Settings fonctionne** mais utilise localStorage au lieu de l'API backend.

**Impact:**
- Les données du wizard ne sont pas affichées
- Les modifications ne sont pas persistées côté serveur
- Pas de sync multi-device

**Recommandation:** Intégrer l'API `/tenant-context/profile/:tenantId` dans une prochaine itération.
