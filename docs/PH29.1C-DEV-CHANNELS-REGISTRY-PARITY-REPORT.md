## Root Cause

**Le code source dans Git était obsolète.**

La PROD a été buildée à partir d'une version du code incluant l'intégration Octopia complète, mais cette version n'a jamais été commitée dans Git (ou a été perdue lors d'un reset).

## Correctif appliqué (DEV)

### 1. Création du service Octopia

**Fichier:** `src/services/octopia.service.ts`

```typescript
export interface OctopiaStatus {
  connected: boolean;
  sellerId?: string;
  mode?: string;
  lastSync?: string;
  ordersCount?: number;
  delegationInstructions?: string;
  apiCredentialsUrl?: string;
}

export async function getOctopiaStatus(tenantId: string): Promise<OctopiaStatus>
export async function connectOctopia(params: {...}): Promise<OctopiaStatus>
export async function disconnectOctopia(tenantId: string): Promise<boolean>
export async function syncOctopia(tenantId: string, days?: number): Promise<{ordersCount: number}>
```

### 2. Mise à jour de channels/page.tsx

- Import du service Octopia
- Ajout des states `octopiaStatus`, `octopiaLoading`
- Fetch du status Octopia au chargement
- Section UI Octopia (Cdiscount) complète
- Retrait de Cdiscount de `otherChannels` (plus "coming_soon")
