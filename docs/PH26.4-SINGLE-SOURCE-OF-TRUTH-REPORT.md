## 3. Requetes SQL finales

### Conversations (source unique)
```sql
SELECT 
  COUNT(*)::int as total,
  COUNT(*) FILTER (WHERE status = 'open')::int as open,
  COUNT(*) FILTER (WHERE status = 'pending')::int as pending,
  COUNT(*) FILTER (WHERE status = 'resolved')::int as resolved,
  COUNT(*) FILTER (WHERE status = 'resolved' AND updated_at > NOW() - INTERVAL '24 hours')::int as resolved_24h,
  COUNT(*) FILTER (WHERE status = 'resolved' AND updated_at > NOW() - INTERVAL '7 days')::int as resolved_7d
FROM conversations WHERE tenant_id = $1
```

### SLA (source unique)
```sql
SELECT 
  COUNT(*) FILTER (WHERE LOWER(sla_state) = 'ok' OR sla_state IS NULL)::int as ok,
  COUNT(*) FILTER (WHERE LOWER(sla_state) = 'at_risk')::int as at_risk,
  COUNT(*) FILTER (WHERE LOWER(sla_state) = 'breached')::int as breached,
  COUNT(*)::int as total
FROM conversations WHERE tenant_id = $1 AND status IN ('open', 'pending')
```

### Orders (source unique)
```sql
SELECT 
  COUNT(*)::int as total,
  COUNT(*) FILTER (WHERE status IN ('Pending', 'Unshipped'))::int as pending,
  COUNT(*) FILTER (WHERE status = 'Shipped')::int as shipped,
  COUNT(*) FILTER (WHERE status = 'Canceled')::int as cancelled,
  COUNT(*) FILTER (WHERE status = 'Delivered')::int as delivered,
  COUNT(*) FILTER (WHERE delivery_status = 'shipped')::int as in_transit,
  COUNT(*) FILTER (WHERE delivery_status = 'delayed')::int as delayed,
  COUNT(*) FILTER (WHERE delivery_status = 'preparing')::int as preparing
FROM orders WHERE tenant_id = $1
```

### SAV Open (orders avec conversations actives)
```sql
SELECT COUNT(DISTINCT o.id)::int as sav_open
FROM orders o
INNER JOIN conversations c ON c.order_ref = o.external_order_id AND c.tenant_id = o.tenant_id
WHERE o.tenant_id = $1 AND c.status IN ('open', 'pending')
```

### Messages (source unique)
```sql
SELECT 
  COUNT(*)::int as total,
  COUNT(*) FILTER (WHERE m.created_at > NOW() - INTERVAL '24 hours')::int as last_24h,
  COUNT(*) FILTER (WHERE m.created_at > NOW() - INTERVAL '7 days')::int as last_7d
FROM messages m INNER JOIN conversations c ON m.conversation_id = c.id
WHERE c.tenant_id = $1
```
