# PH35.3B — Octopia Anti-Race Lock (DEV only)

## Objectif
Eviter les doublons causes par backfill + sync simultanes sur un meme tenant.

## Implementation
`pg_try_advisory_lock(hashtext(tenantId || '_octopia_import'))` dans :
- `backfillOctopiaDiscussions()` 
- `syncOctopiaDiscussions()`

Si lock non obtenu → log `SKIP_LOCKED`, return immediat. Lock relache dans `finally`.

Ajout du flag `OCTOPIA_OUTBOUND_ENABLED` (env var, default `true`). Si `false`, `determineProvider('octopia')` retourne `'mock'` au lieu de `'octopia'`.
