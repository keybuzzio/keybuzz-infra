# PH151.1 — Inbox UX Declutter & Sidebar Intelligence

**Date** : 13 avril 2026
**Environnement** : DEV uniquement
**Image** : `v3.5.62-ph151.1-inbox-clean-dev`
**Rollback** : `v3.5.61-ph151-inbox-ux-intelligence-dev`

---

## Objectif

Corriger PH151 pour supprimer les doublons visuels, centraliser le contexte dans la sidebar droite, ajouter une sidebar collapsible, et nettoyer les couleurs (réduction du violet).

---

## Changements effectués

### 1. Suppression du doublon AICaseSummary (centre)

- **Avant** : `AICaseSummary` affiché dans la zone centrale des messages ET dans la sidebar → doublon visuel
- **Après** : `AICaseSummary` retiré du centre, uniquement dans la sidebar via `topSlot`
- **Fichier** : `app/inbox/InboxTripane.tsx` — suppression du bloc `<AICaseSummary>` dans la zone messages

### 2. Sidebar = source unique de contexte

- **Avant** : Sidebar affichait uniquement "Commande" (OrderSidePanel)
- **Après** : Header renommé "Contexte", résumé IA affiché en premier via `topSlot` prop
- **Fichier** : `src/features/inbox/components/OrderSidePanel.tsx`
  - Ajout props `topSlot?: React.ReactNode` et `collapsible?: boolean`
  - 3 blocs de rendu modifiés : empty state (pas d'orderId), no-order (avec topSlot), et main (avec order)
  - Le résumé IA est toujours visible, même sans commande liée

### 3. Fix statut dynamique

- **Analyse** : Le `handleStatusChange()` utilise un optimistic update correct (setSelectedConversation + setConversations)
- **Fix** : Suppression du `StatusBadge` doublon dans `ConversationSummaryBar` qui faisait redondance avec le dropdown de statut dans le header
- **Le dropdown de statut reste la source unique** : il met à jour l'API et le state React en temps réel

### 4. Sidebar collapsible

- **Ajout** : Bouton "Réduire" (icône PanelRightOpen) dans le header de la sidebar
- **Collapse** : Sidebar réduite à une barre de 10px avec un bouton "Développer" (PanelLeftOpen)
- **État** : State React local (`collapsed`), pas de localStorage
- **Fonctionne** : dans tous les états (empty, no-order, with-order)

### 5. Nettoyage visuel

| Élément | Avant (PH151) | Après (PH151.1) |
|---|---|---|
| AICaseSummary couleurs | `purple-*` (violet intrusif) | `gray-*` (neutre), `amber-500` icône |
| ConversationSummaryBar | Background `bg-gray-50/80`, StatusBadge | Inline flex, pas de background, pas de badge |
| Bouton "Ouvrir commande" | `bg-purple-600` | `bg-gray-700` |
| Icône ShoppingBag | `text-purple-500` | `text-gray-500` |
